# spotify clone
A static web page inspired by Spotify, built using HTML and CSS.

## Description
This project is a web-based music player interface inspired by Spotify. 
It includes a sidebar, library section, recently played cards, trending section, and a music player layout at the bottom. 
This project focuses on UI design using HTML and CSS.

## Features
- Sidebar navigation (Home, Search, Your Library)
- Library section with playlist cards
- Recently played and trending cards
- Music player layout at the bottom
- Styled using CSS and Google Fonts
- Uses Font Awesome icons

## Tech Stack
- HTML5
- CSS3
- Font Awesome icons
- Google Fonts (Montserrat)
